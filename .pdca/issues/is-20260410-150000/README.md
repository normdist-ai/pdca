## 问题背景

模板文件放在`/.pdca/`文件夹中不合适。按SIPOC方法思考：
- `/.pdca/`文件夹是PDCA技能的**输出**
- 模板文件是**过程规范**

## 问题影响

- 目录结构不符合SIPOC方法论
- 过程规范和输出文件混淆
- 不利于理解和使用

## 预期目标

- 将模板文件从`.pdca/templates/`移动到`.trae/skills/pdca/templates/`
- 明确区分过程规范和输出文件
- 符合SIPOC方法论
