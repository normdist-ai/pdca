# AgentSkills 技能描述规范

## 核心原则

### 技能的使用者

**重要**：技能的使用者是AI代理（Agent），不是最终用户！

- ✅ **正确**：技能指导代理如何帮助用户
- ❌ **错误**：技能引导用户如何工作

### 描述规范

#### frontmatter description 字段

**推荐格式**：
```
[技能功能描述]。指导代理[具体方法]，帮助用户[实现目标]。当用户提到[触发关键词]时自动调用。
```

**示例**：
```yaml
description: 基于ISO9000标准的PDCA循环方法论技能。指导代理按照策划、实施、检查、处置四个阶段系统化开展工作，帮助用户实现持续改进。当用户提到PDCA、持续改进、质量管理、项目策划、问题解决时自动调用。
```

#### 概述部分

**推荐格式**：
```
本技能基于[理论基础]，指导代理如何使用[方法/工具]帮助用户[实现目标]。
```

**示例**：
```
本技能基于ISO9000质量管理体系标准中的PDCA循环理论，指导代理如何使用Plan（策划）、Do（实施）、Check（检查）、Act（处置）四个阶段帮助用户实现持续改进。
```

#### 使用指南部分

**推荐格式**：
```
### 如何使用本技能

代理应按照以下步骤使用本技能：

1. **步骤1**：[代理应该做什么]
2. **步骤2**：[代理应该做什么]
...
```

**示例**：
```
### 如何使用本技能

代理应按照以下步骤使用本技能：

1. **识别当前阶段**：判断用户处于PDCA哪个阶段
2. **提供阶段指导**：给出该阶段的关键活动和输出物
3. **引导下一阶段**：完成当前阶段后，引导用户进入下一阶段
4. **形成闭环**：确保帮助用户完成完整的PDCA循环
```

#### 常用操作示例

**推荐格式**：
```
**[操作名称]**：
```
用户：[用户输入]
代理：[代理响应]
```
```

**示例**：
```
**开始一个PDCA循环**：
```
用户：我想用PDCA方法改进[项目/流程/问题]
代理：按照PDCA四阶段帮助用户开展工作
```
```

## 常见错误

### 错误1：混淆技能使用者

❌ **错误**：
```
引导用户按照PDCA四阶段开展工作
```

✅ **正确**：
```
指导代理按照PDCA四阶段帮助用户开展工作
```

### 错误2：技能作为主语

❌ **错误**：
```
技能：提供Plan阶段的关键活动、输出物和注意事项
```

✅ **正确**：
```
代理：提供Plan阶段的关键活动、输出物和注意事项
```

### 错误3：省略代理角色

❌ **错误**：
```
1. 识别当前阶段：判断用户处于PDCA哪个阶段
```

✅ **正确**：
```
代理应按照以下步骤使用本技能：

1. 识别当前阶段：判断用户处于PDCA哪个阶段
```

## 检查清单

在编写或审查技能文档时，请检查以下项目：

- [ ] frontmatter description 明确说明"指导代理"
- [ ] 概述部分明确说明"指导代理如何...帮助用户"
- [ ] 使用指南部分明确说明"代理应按照以下步骤使用本技能"
- [ ] 常用操作示例中使用"代理："而不是"技能："
- [ ] 所有描述都遵循"代理使用技能帮助用户"的逻辑
- [ ] 没有将技能的使用者误写为用户
