# PDCA度量指标

## 效果度量指标

| 指标类别 | 具体指标 | 计算方法 | 目标值 |
|----------|----------|----------|--------|
| **目标达成率** | 目标完成度 | (实际结果/目标值)×100% | ≥90% |
| **时间效率** | 循环周期 | Plan+Do+Check+Act总时长 | 按项目设定 |
| **成本效益** | 投入产出比 | (改进收益/改进成本)×100% | ≥150% |
| **质量提升** | 问题解决率 | (已解决问题/总问题数)×100% | ≥85% |
| **持续改进** | 循环次数 | 单位时间内的PDCA循环次数 | 持续增加 |

## 过程度量指标

| 阶段 | 度量指标 | 说明 |
|------|----------|------|
| **Plan** | 计划完整性 | 目标、WBS、资源、风险是否齐全 |
| **Do** | 执行符合度 | 实际执行与计划的符合程度 |
| **Check** | 数据准确性 | 检查数据的准确性和完整性 |
| **Act** | 标准化程度 | 改进措施的标准化和推广程度 |

## 度量指标使用指南

### 1. 目标达成率

**计算方法**：
```
目标达成率 = (实际结果 / 目标值) × 100%
```

**评估标准**：
- ≥90%：优秀
- 80%-89%：良好
- 70%-79%：合格
- <70%：需改进

**应用场景**：
- 项目目标完成度评估
- 改进措施效果评估
- 个人绩效评估

### 2. 时间效率

**计算方法**：
```
循环周期 = Plan时长 + Do时长 + Check时长 + Act时长
```

**评估标准**：
- 按计划完成：优秀
- 延迟<10%：良好
- 延迟10%-20%：合格
- 延迟>20%：需改进

**应用场景**：
- 项目进度管理
- 流程效率评估
- 资源配置优化

### 3. 成本效益

**计算方法**：
```
投入产出比 = (改进收益 / 改进成本) × 100%
```

**评估标准**：
- ≥150%：优秀
- 120%-149%：良好
- 100%-119%：合格
- <100%：需改进

**应用场景**：
- 项目投资回报评估
- 改进措施成本效益分析
- 资源分配决策

### 4. 质量提升

**计算方法**：
```
问题解决率 = (已解决问题数 / 总问题数) × 100%
```

**评估标准**：
- ≥85%：优秀
- 75%-84%：良好
- 65%-74%：合格
- <65%：需改进

**应用场景**：
- 质量改进效果评估
- 问题管理效率评估
- 团队能力提升评估

### 5. 持续改进

**计算方法**：
```
循环次数 = 单位时间内的PDCA循环次数
```

**评估标准**：
- 持续增加：优秀
- 保持稳定：良好
- 略有减少：合格
- 明显减少：需改进

**应用场景**：
- 持续改进文化评估
- 组织学习能力评估
- 改进动力评估

## 度量数据收集方法

1. **定期记录**：每个阶段结束后记录相关数据
2. **数据验证**：确保数据的准确性和完整性
3. **趋势分析**：对比历史数据，分析趋势
4. **可视化展示**：使用图表展示度量结果
5. **定期评审**：定期评审度量指标的有效性

## 度量结果应用

1. **绩效评估**：作为个人和团队绩效评估的依据
2. **改进决策**：为改进决策提供数据支持
3. **资源分配**：优化资源配置
4. **目标设定**：为下一循环设定更合理的目标
5. **经验分享**：分享成功经验和最佳实践
