# 技能使用检查清单

## 使用前检查

- [ ] 是否完整阅读了技能的SKILL.md？
- [ ] 是否理解了技能的工作流程？
- [ ] 是否查看了技能的模板文件？
- [ ] 是否明确了文件的存放位置？

## 执行中检查

- [ ] 是否按照技能的工作流程逐步执行？
- [ ] 是否使用了技能提供的模板？
- [ ] 是否在正确的位置创建文件？
- [ ] 是否完成了所有必需的步骤？

## 执行后检查

- [ ] 是否创建了所有必需的文件？
- [ ] 是否使用了正确的文件名？
- [ ] 是否填写了所有必需的内容？
- [ ] 是否符合技能的规范要求？

## 常见错误

1. **没有完整阅读技能文档** → 必须完整阅读SKILL.md
2. **跳过工作流程步骤** → 必须按照工作流程逐步执行
3. **不使用标准模板** → 必须使用技能提供的模板
4. **文件位置错误** → 必须思考文件应该放在哪里
